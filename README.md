# Sitios Históricos Argentina

Aplicación web full-stack para la exploración y gestión de sitios históricos de Argentina.

La plataforma permite a los usuarios:
- Explorar sitios históricos geolocalizados
- Visualizar información detallada
- Calificar sitios
- Publicar reseñas
- Gestionar contenido multimedia

## Arquitectura

La aplicación está compuesta por:

- Frontend desarrollado en Vue.js
- API REST desarrollada en Flask
- PostgreSQL + PostGIS para persistencia y consultas geoespaciales
- MinIO para almacenamiento de contenido multimedia
- Contenedores Docker orquestados mediante Docker Compose

### Infraestructura
- Docker

Proyecto académico desarrollado en la Facultad de Informática de la UNLP.
